Exercice – Astro + TypeScript

Exercice

Créer une application web servie avec Astro, en TypeScript qui affiche : "Bonjour, TypeScript!" dans la page web.

Étapes à suivre :

  1. Créer programmétiquement (depuis le script TypeScript) un <div> avec un id="app".
  2. Créer un <h1> avec le texte "Bonjour, TypeScript!".
  3. Ajouter le <h1> dans le <div>.
  4. Ajouter le <div> dans le <body>.
  5. Démarrer un serveur avec Astro.
  6. Déboguer dans Chrome votre application.
