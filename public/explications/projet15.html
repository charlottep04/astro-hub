Exercice – Playlist Deezer

Exercice

Faites une application qui s'exécute côté SERVEUR (pour ne pas être bloqué par CORS) et qui affiche une liste des titres de la playlist "top 100 France 2023" disponible à l'adresse suivante : https://api.deezer.com/playlist/11846226041

Étapes à suivre :

  1. Créez une application serveur pour récupérer les données de la playlist Deezer. Utilisez un framework côté serveur (comme Express pour Node.js) afin de contourner le problème du CORS.
  2. Récupérez les informations nécessaires pour chaque chanson dans la playlist, notamment : le titre, l'artiste, l'album, l'album cover (petite image), et un extrait audio en format mp3.
  3. Affichez les résultats sous forme de liste non ordonnée (<ul>) où chaque chanson est un élément de liste (<li>).
  4. Pour chaque chanson, incluez les informations suivantes :
    • Le titre de la chanson
    • Le nom de l'artiste
    • Le titre de l'album
    • Une image de l'album (<img>)
    • Un lecteur audio pour écouter un extrait (<audio>)
  5. Assurez-vous que l'attribut preload="none" est utilisé dans la balise <audio> afin d'éviter de télécharger les extraits de chansons au démarrage.
