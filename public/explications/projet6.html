Consignes :

  1. Créer un dossier nommé 06_CSS_flexbox dans votre dépôt Git.
  2. Pour chaque exercice, créer un fichier exX.html et un fichier exX.css (où X = 1 à 4).
  3. Relier les fichiers CSS à leurs fichiers HTML respectifs avec une balise <link>.
  4. Faire un commit et pousser le dossier complet sur votre dépôt Git.
